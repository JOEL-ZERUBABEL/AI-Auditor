# 🚀 AI Decision Boardroom

AI Decision Boardroom is a Multi-Agent AI application built using Streamlit, Groq LLMs, and Pydantic.

The platform simulates a boardroom where multiple AI agents analyze a user's startup, investment, or career decisions and provide structured recommendations.

---

## 🎯 Features

### Startup Boardroom
- Startup Analysis Agent
- Industry Evaluation
- Budget Analysis
- Team Size Evaluation
- CEO Decision Agent
- Action Plan Generation
- Timeline Recommendations

### Investment Boardroom
- Investment Advisor Agent
- Risk Analysis Agent
- Final Verdict Agent
- Market Outlook Analysis
- Growth Potential Assessment
- Investment Recommendations

### Career Boardroom
- Career Advisor Agent
- Skill Analysis
- Market Demand Evaluation
- Career Recommendations
- Learning Roadmap Suggestions

## 🤖 AI Agents

### Startup Agent
Analyzes:
- Startup Idea
- Industry
- Budget
- Team Strength

Outputs:
- Startup Report
- Strategic Recommendations

### CEO Agent
Reviews:
- Startup Report

Outputs:
- CEO Score
- Approval Percentage
- Timeline
- Action Items

### Investment Advisor
Analyzes:
- Asset
- Investment Amount
- Duration
- Risk Tolerance

Outputs:
- Growth Potential
- Market Outlook
- Investment Recommendation

### Risk Analyst
Evaluates:
- Market Risks
- Economic Risks
- Volatility

Outputs:
- Risk Report

### Verdict Agent
Combines:
- Investment Report
- Risk Report

Outputs:
- Final Investment Decision

### Career Advisor
Analyzes:
- Courses
- Interests
- Current Market Demand

Outputs:
- Career Recommendations
- Future Opportunities
- Learning Suggestions

## ⚙️ Installation

Clone Repository

```bash
git clone https://github.com/yourusername/ai-decision-boardroom.git
cd ai-decision-boardroom
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Create Environment File

```env
GROQ_API_KEY=YOUR_API_KEY
```

Run Application

```bash
streamlit run frontend.py
```
